guia basica de HTML

introduccion

HTML (hipertext markut languaje) es el lenguaje estandar para hacer paginas web.

en esta clase vamos a aprender el uso de las etiquetas basicas como:

Heading <h1> : para titulos y subtitulos.

Parrafo <p> : para parrafos.

Div <div> : para agregar ciontenido.

Ejercicio

vamos a realizar ejercicio de practica en el cual cuenten lo que ustedes quieran siguiendo las intrucciones de abajo.

Siga las siguientes instrucciones para realizarlo.

1-Crea un Titulo principal usando una etiqueta <h1>

2- Agrega al menos dos subtitulos utilizando <h2> o <h3>

3- escribe un nuevo parrafo para cada subtitulo utilizando la etiqueta <p>

4- Envuelve cada seccion en un contenedor utilizando la etiqueta <div>
